## List JSON-RPC

### Get Block Number

```type=json
{
        "jsonrpc":"2.0",
        "method":"eth_blockNumber",
        "params":[],
        "id":1
}
```
### Get TransactionByHash

```type=json
{
    "jsonrpc": "2.0",
    "method": "eth_getTransactionByHash",
    "params": [
        "0xf57bd45154c6bbcf2a4be2e36ae0db40a2555422b5cb712a593d3555539ea941"
    ],
    "id": 3
}
```
